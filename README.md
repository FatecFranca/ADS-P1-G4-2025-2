A ONG realiza um trabalho incrível de formação de caráter com crianças, mas a gestão era 100% manual. Fichas de papel, controle de presença em cadernos e dificuldade de acompanhar a evasão escolar eram os maiores gargalos. O risco de perda de dados e a sobrecarga administrativa dos voluntários eram problemas críticos.
Por conta disso criamos um sistema de controle de presença/chamada, para que os voluntários da ONG consigam gerenciar melhor as aulas.
